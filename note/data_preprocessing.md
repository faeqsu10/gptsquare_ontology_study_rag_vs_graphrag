: 각 row를 줄글로 변환 -> 벡터db 저장

처음에는 데이터 사이즈가 작아서
식당 정보와 같이 마크다운 변환해서 벡터db에 저장 하였지만
같은 테이블로 구성된 정보이기 때문에
각 row를 줄글로 변환하여 벡터db에 저장하는 동일한 방법으로 진행


기본 정보 - 식당 정보

2.데이터_직원
3.데이터_공급업체
